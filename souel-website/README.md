## How to Use

1. **Open the Website**: Simply open `index.html` in any web browser
2. **Navigate**: Use the navigation menu to jump to different sections
3. **View Menu**: Scroll to the menu section and use filters to browse items
4. **Contact**: Fill out the contact form to send messages

## Customization Guide

### Change Cafe Information
Edit `index.html` and update:
- Address (in Contact section)
- Phone number
- Email address
- Business hours

### Update Menu Items
In the Menu section of `index.html`:
- Add/remove menu items
- Update prices
- Change category (data-category attribute)
- Replace emoji icons with images

### Modify Colors
Edit `assets/css/style.css` and change the CSS variables at the top:
```css
:root {
    --primary-color: #8B5A3C;      /* Main brown */
    --secondary-color: #D4A574;    /* Gold accent */
    --accent-color: #F5E6D3;       /* Light cream */
    --dark-color: #2C1810;         /* Dark brown */
    --light-color: #F9F6F0;        /* Light background */
}
```

### Add Real Images
Replace emoji icons with actual images by modifying the `.item-image` divs in the menu items.
